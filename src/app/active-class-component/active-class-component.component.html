active-class 測試


靜態寫法:

[ngClass]版本

  • ngClass 用 string

  • ngClass 用 array

  • ngClass 用 object

  • ngClass 用 object 配字串

[class]版本

  • [class]指定一種 class 寫法

  • [class]用 object 配字串(無效果寫法)

  • [class] 用 string

  • [class] 用 array

  • [class] 用 object

動態寫法:

[ngClass]版本

  • 顯示與不顯示 用 string 控制

  • 顯示與不顯示 用 object 單一 class 控制

  • 顯示與不顯示 用 function return

  • 顯示與不顯示 用 object 多 class 控制一個

  • 顯示與不顯示 用 object 對 object 內屬性作切換 (無效寫法)

  • 測試顏色切換 綁定 function return string

  • 測試顏色切換 綁定 function return array

  • 測試顏色切換 綁定 function return object

[class]版本

  • 顯示與不顯示 用 string 控制

  • [class] 單一 class 綁定變量寫法

  • [class] 綁定 return function

  • [class] 用 object

  • [class] 用 object 多 class 控制一個

  • 測試顏色切換 綁定 function return string

  • 測試顏色切換 綁定 function return array

    消除顏色無法變化是因為深層變化無法被監聽到

  • 測試顏色切換 綁定 function return object

    消除顏色無法變化是因為深層變化無法被監聽到

class 版本

  • class 用模板語法 return function
  • class 用模板語法 return function 複製變量
